# Poetry Installation

- [Install pipx](https://pipx.pypa.io/stable/installation/)
- Install poetry: `pipx install poetry`
- Initialize Poetry project: `poetry install`
- Activate virutal env: `poetry shell`

- Initialize Poetry project: `poetry init`
- Install dependencies: `poetry add flask clickhouse-connect python-dotenv`
- Install dev dependencies: `poetry add --dev pytest black flake8 mypy isort`
- Activate virutal env: `poetry shell`
